# 🔬 MeddyNet Lab Partner Portal
### The Premium Diagnostic Operating System for Modern Laboratories

MeddyNet Lab Partner Portal is a high-performance, standalone SaaS platform designed for diagnostic centers to manage their entire operational lifecycle. Built with a "Mobile-First, Premium-Always" philosophy, it offers a seamless blend of analytical intelligence and operational efficiency.

---

## 🚀 Key Modules & Experience

### 📊 Tactical Dashboard
- **Real-time Analytics**: Instant visibility into daily revenue, booking volume, and growth trends.
- **Operational Intelligence**: Smart alerts for delayed reports and urgent sample collections.
- **Recent Activity**: A live feed of the latest bookings and status transitions.

### 📅 Advanced Service Management
- **Interactive Calendar**: A high-fidelity timeline view for tracking home collections and lab visits with precision.
- **Smart Bookings**: Comprehensive lifecycle management from "Sample Collected" to "Result Delivered".
- **Test Menu Controller**: Granular control over test pricing, categories, and turnaround times (TAT).

### 🧪 Technical Operations
- **Report Upload Center**: A secure, HIPAA-ready interface for lightning-fast result delivery with simulated progress tracking.
- **Technician Management**: Phlebotomist onboarding, real-time status tracking (On-Duty/Idle), and rating systems.
- **Profile Hub**: Full business identity management, including NABL accreditation and facility listing.

### 💰 Financial Ecosystem
- **Earnings Console**: Professional revenue analysis with monthly growth patterns and commission breakdowns.
- **Payout Ledger**: Transparent transaction history with detailed itemization for every settlement.

## 🛠️ Tech Stack (The Next-Gen Suite)

- **Framework**: [Next.js 15+](https://nextjs.org/) (App Router & Server Actions)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/) (Ultra-modern configuration)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Language**: TypeScript

## 🎯 Features

- Real-time analytics and operational intelligence
- Advanced service management with interactive calendar and smart bookings
- Technical operations management with report upload center and technician management
- Financial ecosystem with earnings console and payout ledger
- Mobile-first design with premium user experience

## 📦 Installation

To install the project, run the following command:
```bash
npm install
```
or
```bash
yarn install
```

## 🚀 Usage

To start the development server, run the following command:
```bash
npm run dev
```
or
```bash
yarn dev
```
Open your browser and navigate to `http://localhost:3000` to access the application.
